# タイピングモンスターバトル

ローマ字をタイピングしてモンスターを倒そう!
小学校低学年向けのタイピング練習ゲームです。

## 遊び方

### ゲームを始める

チャットで以下のコマンドを入力してね:

| コマンド                | 説明                              |
| ----------------------- | --------------------------------- |
| `start` または `start1` | かんたんモード（ひらがな 1 文字） |
| `start2`                | ふつうモード（かんたんな単語）    |
| `start3`                | むずかしいモード（長い単語）      |

### ゲームの進め方

1. ゲームを開始する（`start`と入力）
2. モンスターが現れると、ひらがなが表示されます
3. **そのままローマ字を入力!**
   - 例:「いぬ」→ `inu`
   - 例:「りんご」→ `ringo`
4. 正解するとモンスターを倒せるよ!

### コンボシステム

連続で正解するとコンボボーナスがもらえるよ!

### その他のコマンド

| コマンド | 説明               |
| -------- | ------------------ |
| `score`  | 現在のスコアを確認 |
| `stop`   | ゲーム終了         |
| `help`   | ヘルプを表示       |

## 難易度について

### かんたん（レベル 1）

- ひらがな 1 文字のローマ字
- 例:「あ」→ `a`、「か」→ `ka`

### ふつう（レベル 2）

- かんたんな単語
- 例:「いぬ」→ `inu`、「りんご」→ `ringo`

### むずかしい（レベル 3）

- 長めの単語やマインクラフト用語
- 例:「ぞんび」→ `zonbi`、「ばなな」→ `banana`

## ファイル構成

| ファイル       | 説明                       |
| -------------- | -------------------------- |
| `gameState.ts` | ゲーム状態管理             |
| `questions.ts` | 問題データと出題ロジック   |
| `gameLogic.ts` | メインゲームロジック       |
| `commands.ts`  | チャットコマンドハンドラー |
| `main.ts`      | エントリーポイント         |

## MakeCode での使い方

1. MakeCode for Minecraft Education Edition を開く
2. 「読み込む」→「ファイルを読み込む」を選択
3. このプロジェクトフォルダを選択
4. Minecraft に接続してプレイ!
